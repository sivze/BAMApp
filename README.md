# BAMApp
A cross-platform application for retail consumer survey. 

# Project Objectives
### Build a cross-platform mobile application:
•	Pushes store-specific survey notifications to app users when they enter a participating store.
•	Integrates with Google Maps and Survey Monkey to push the alerts.
•	Generates one-time use, nontransferable coupons once the survey is complete.
•	Alerts the admin user each time there is a survey completed.
